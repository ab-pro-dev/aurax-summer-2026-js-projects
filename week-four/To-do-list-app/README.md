# To-Do List App

A simple task manager with local storage persistence, built with HTML, CSS, and JavaScript.

## Features

- Add, edit, and delete tasks
- Mark tasks as completed (click text to toggle)
- Clear all completed tasks at once
- Task counter shows remaining items
- Fade-in/out animations
- Data persists across sessions via localStorage

## How to Use

1. Open `index.html` in a browser
2. Type a task and press Enter or click **Add**
3. Click task text to toggle completion
4. Click ✎ to edit, × to delete

## Files

- `index.html` — App layout
- `style.css` — Styling
- `script.js` — App logic
