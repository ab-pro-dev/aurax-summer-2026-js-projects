# Tic-Tac-Toe

A two-player Tic-Tac-Toe game built with HTML, CSS, and JavaScript.

## How to Play

1. Open `index.html` in a web browser
2. Player X goes first
3. Click an empty cell to place your mark
4. Players alternate turns (X and O)
5. First player to get 3 in a row wins
6. Click **Restart** to play again

## Features

- 3×3 game board
- Turn-based gameplay (X and O)
- Win detection with highlighted winning cells
- Draw detection
- Restart button
- Responsive design

## Files

- `index.html` - Game structure
- `style.css` - Styling
- `script.js` - Game logic
